# O que é Golang?

O Go é uma linguagem criada pelo time do Google em 2009. Ela foi desenvolvida para resolver problemas de sistemas grandes, onde linguagens como Java e C++ estavam ficando complexas demais.

# O objetivo do Go é ser:

Simples de aprender,
Fácil de manter,
Rápido para executar,
Excelente para concorrência,
Fácil de implantar em produção.

Por que Go foi criado?

# Os engenheiros do Google precisavam lidar com:

Servidores gigantes,
Milhares de requisições simultâneas,
Compilação rápida,
Código simples de manter.

# Por isso eles criaram uma linguagem que mistura características de:

Linguagem	O que Go herdou,
C	Simplicidade e velocidade,
Python	Facilidade de leitura,
Java	Organização e tipagem,
Erlang	Concorrência.

Onde Go é utilizado atualmente?

# Muitas ferramentas modernas foram desenvolvidas em Go:

Docker,
Kubernetes,
Terraform,
Prometheus,
Grafana Loki.

# Por isso Go é extremamente valorizado em:

Cloud Computing,
DevOps,
Backend,
Microserviços,
APIs,
Sistemas distribuídos,
Ferramentas de infraestrutura.
